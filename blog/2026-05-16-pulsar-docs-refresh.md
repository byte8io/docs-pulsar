The Pulsar docs have a fresh look. Same content, new chrome — aligned with the Orbit docs and the wider byte8.io marketing aesthetic.

<!-- truncate -->

## What changed

- **Visual style** matches the byte8.io marketing site — deep dark background, soft rounded surface panels, hairline borders, purple accent (`#8B5CF6`) sourced from the dashboard primary colour.
- **Homepage rewritten** with a hero + stats strip + feature-card sections + CTA band — same pattern as [docs.byte8.io/orbit](https://docs.byte8.io/orbit/) and the byte8.io product pages.
- **Navbar** now includes a GitHub link to [`byte8io/module-pulsar`](https://github.com/byte8io/module-pulsar), a Pricing link, and a Dashboard CTA pill on the right.
- **Changelog** lives at [/blog](/blog) — release notes will land here from now on.

## What didn't change

The doc content itself is unchanged — quick start, HTTP checks, checkout flows, alert integrations, the GraphQL API reference, the Magento extension health collectors. Same words, just framed in the new shell.

## What's coming

- **More monitoring docs** — synthetic transactions beyond checkout (login flow, admin reachability)
- **Algolia DocSearch** — once we cross the threshold where local search isn't enough
- **More changelog posts** — release notes for the Pulsar Magento extension are catching up here

## Found a doc bug?

Each page has an **Edit this page** link in the footer that drops you straight into the right file on GitHub. PRs welcome.
